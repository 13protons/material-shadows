Material Shadow

A stylesheet to produce shadows on html elements that create subtle depth cues. Inspired by google's material design.

Basic Usage

Add a class of shadow-[number], where [number] is an integer from 1-10 that represents an object's distance from it's ground plane.

.shadow-1
.shadow-3
.shadow-5

Add movement

You can specify an 'active' shadow that is transitioned to when an element is being :hovered over. Add this as another class, shadow-active-[number]

Hover over these elements to see the effect:

.shadow-2
.shadow-active-5
.shadow-2
.shadow-active-7
